### 프로그램구성

객실예약: 

객실 호수와 등급, 가격을 조회 → 객실테이블 조회

어떤 호수를 선택할지, 이름입력

예약취소:

취소하면 예약 테이블 요소 delete

객실상태조회:

호수,등급,예약자명 조회

+각자 추가하고 싶은 요소(원하면)

### 사용기술

Java 1.8

Oracle 10

JDBC14

입력: Scanner

출력: 콘솔 출력창

### 제작기간: 일주일

1일차 - 다함께 구체적인 db설계하기(ERD 사용)

2일차~6일차 - 각자 어느 부분을 만들었는지 매일 간단하게 보고

7일차 - 각자 프로그램 시연 혹은 실패후 막힌 부분 공유
