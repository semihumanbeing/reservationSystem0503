# 미니프로젝트[호텔예약] 기획서

자바와 JDBC api를 사용해서 콘솔 프로그램을 만드는 미니프로젝트

## 목표

1. JDBC 사용하여 DB와 프로그램 연동
2. JAVA→ DB 로 CRUD 구현하기

## 주제

호텔 예약 관리 시스템 만들기

### 프로그램구성
